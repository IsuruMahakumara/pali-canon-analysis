# PostgreSQL with Apache AGE + PGVector

This repository provides a **custom Docker image** of PostgreSQL 16 with:

- **Apache AGE** (graph database extension)
- **PGVector** (vector similarity extension)

It is designed for **graph + vector workloads** and is ready to use with Docker or Docker Compose.

---

## Features

- PostgreSQL 16
- AGE and PGVector extensions pre-installed
- Automatic initialization of extensions on first container start
- Persistent database storage via Docker volumes
- Easy to rebuild or update extensions in the container
